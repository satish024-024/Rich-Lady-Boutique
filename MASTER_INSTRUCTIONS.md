# Rich Lady Boutique — Master Project Instructions (Read Before Writing Any Code)

## Project Objective

Build a world-class luxury boutique eCommerce website for **Rich Lady Boutique**.

This is **NOT** a normal fashion website.

It must feel like an editorial luxury brand experience that combines storytelling, handcrafted aesthetics, premium typography, subtle motion, and modern ecommerce.

The inspiration should come from brands such as:

- Apple
- Aesop
- COS
- Loewe
- Sabyasachi
- Zara Studio
- Ralph Lauren
- Celine

Never copy them.

Instead, understand why their websites feel premium and apply those design principles.

The final result should be portfolio quality.

# Self Review Requirement

After completing each phase:

Review your own work as if you were a Senior UI/UX Designer at Apple.

Ask yourself:

- Does this look premium?
- Does it look handcrafted?
- Does it feel like a template?
- Is there enough whitespace?
- Is the typography elegant?
- Are animations subtle?
- Would this impress a Dribbble featured designer?
- Is every section visually unique?

If any answer is "No",

Improve the implementation before moving to the next phase.

Never stop at the first acceptable solution.

Iterate until the design feels exceptional.
